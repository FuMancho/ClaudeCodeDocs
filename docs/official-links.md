# Official Documentation & Resources

This document provides a central directory for all official Anthropic resources related to Claude and the Claude Code CLI. Always refer to these verified sources for the most accurate and up-to-date information.

## Core Documentation

- **[Claude Code Official Site](https://code.claude.com)**
  The primary entry point for Claude Code documentation, installation instructions, and CLI reference.

- **[Anthropic Documentation](https://docs.anthropic.com)**
  General documentation for Anthropic's models, APIs, and developer tools. This includes detailed guides on prompt engineering and model capabilities.

- **[Anthropic Console / Platform](https://platform.claude.com)**
  The developer console where you can manage API keys, monitor usage, and explore the API Workbench.

## Why Use Official Links?

Official resources ensure that you are following the latest security protocols and architectural best practices. Community-maintained docs (like this repository) strive for accuracy, but the domains listed above are the ultimate sources of truth.
